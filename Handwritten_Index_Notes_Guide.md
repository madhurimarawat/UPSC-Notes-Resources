# 📝 Handwritten Index Notes Guide

Each index PDF is named as: `Subject_Name_Handwritten_Index`.  
It contains a **complete list of topics along with page numbers**, helping you quickly understand what the notes include.

The corresponding notes are named: `Subject_Name_Handwritten_Notes`.

---

## 📂 About the Index

The index is kept **separately from the main notes** to make navigation easier.  
This allows you to:
- Quickly scan available topics  
- Jump directly to required pages  
- Easily add more indexes in the future  

> [!TIP]
> Always check the index first before opening the notes, it saves time and effort.

---

## 📘 Notes

All pages in the notes are **properly numbered**, and these numbers match the index for easy navigation.

The notes are created using **colored pens** to:
- Improve readability  
- Clearly separate different topics  
- Make studying more engaging  

> [!IMPORTANT]
> Page numbers in the index and notes are aligned, use them to navigate efficiently.

---

## 🚀 How to Use

1. Open the **Index PDF**  
2. Find your topic and note its **page number**  
3. Open/download the **Notes PDF**  
4. Jump directly to the required page  

> [!NOTE]
> This method helps you study faster and stay well-organized.
