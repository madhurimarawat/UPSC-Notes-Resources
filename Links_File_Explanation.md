# 📄 Links.txt Explanation

Instead of uploading PDFs directly, I’ve provided a `Links.txt` file containing relevant resource links. This approach is intentional for the following reasons:

> [!TIP] 
> 
> **Why use links instead of PDFs?**
> * To **credit original authors** and support their work by driving traffic to the source
> * To **avoid unnecessary rehosting** when the content is already available online
> * To **respect copyright**, as some materials are meant for personal use only and not for redistribution

---

## 🚀 How to Use

> [!NOTE]
>
> 1. Open the `Links.txt` file
> 2. Copy the desired link
> 3. Paste it into your browser
> 4. Download or view the content directly from the source

You can also rename downloaded files using the titles provided above each link for better organization.

---

## 📌 Example

```txt
Characters:
https://www.sparknotes.com/lit/flies/characters/
```

> [!TIP]
> This link contains character analysis for *Lord of the Flies*.
> The text above the link acts as a **label/description** for easy reference.

---

## 📂 Additional Note

> [!WARNING]
> If you find a `Links.txt` file inside any folder, it contains **all relevant resources for that topic**.
> The heading above each link indicates the content, which you can reuse as a filename or modify as needed.
