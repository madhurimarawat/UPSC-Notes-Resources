## 📄 Markdown (.md) to PDF Conversion

Some files in this repository are in **Markdown format** (with the `.md` extension). These files display well on GitHub, but may not open properly on all devices (such as phones or basic document viewers).

To view them easily offline, you can convert them into PDF format using the following tool:

🔗 https://md2pdf.netlify.app/

### 🔧 How to Use
- Copy and paste the Markdown content into the website, **or**
- Upload the `.md` file directly
