## 🌍 **Human Evolution + Geological Time Quick Chart**

| **Era / Period**                     | **Epoch (if key)**           | **Approx. Time (MYA = million years ago)** | **Major Events / Significance**                                                        |
| ------------------------------------ | ---------------------------- | ------------------------------------------ | -------------------------------------------------------------------------------------- |
| **Precambrian (Hadean–Proterozoic)** | —                            | 4600 – 540 MYA                             | 🌋 Earth forms → oceans 🌊 → first life (bacteria, algae) 🌱                              |
| **Paleozoic**                        | —                            | 540 – 250 MYA                              | Marine life 🐚 → fishes 🐠 → amphibians 🐸 → plants colonize land 🌿                       |
| **Mesozoic (Age of Reptiles)**       | —                            | 250 – 65 MYA                               | 🦖 Dinosaurs dominate → mammals & birds evolve → extinction event                       |
| **Cenozoic (Age of Mammals)**        | **Tertiary (65 – 2.6 MYA)**  | 65 – 2.6 MYA                               | Mammals diversify 🐘 → primates evolve 🐒 → *Australopithecus* appears                   |
|                                      | **Quaternary → Pleistocene** | 2.6 MYA – 11,700 yrs ago                   | ❄️ Ice Ages → *Homo habilis*, *erectus*, *Neanderthal* evolve → tool cultures develop 🔨 |
|                                      | **Holocene (Current Epoch)** | 11,700 yrs – Present                       | 🌾 Neolithic Revolution → agriculture, civilization, modern *Homo sapiens*              |

### 🧬 **Human Evolution Snapshot**

| **Species**             | **Time Range**    | **Key Traits**                            | **Tools / Culture**        |
| ----------------------- | ----------------- | ----------------------------------------- | -------------------------- |
| *Australopithecus*      | ~4 – 2 MYA        | Upright posture, small brain (400–500 cc) | Pebble tools (rudimentary) |
| *Homo habilis*          | 2.4 – 1.6 MYA     | “Handy man”, larger brain (600–700 cc)    | Oldowan tools              |
| *Homo erectus*          | 1.8 MYA – 0.3 MYA | Fire 🔥, hunting, shelters                 | Acheulean hand-axes        |
| *Homo neanderthalensis* | 0.3 – 0.04 MYA    | Burial practices ⚰️, language hints        | Mousterian tools           |
| *Homo sapiens (modern)* | 0.3 MYA – Present | Symbolic thought, art 🎨, agriculture      | Upper Paleolithic tools    |
